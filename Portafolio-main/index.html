Sobre Mí

¿Quién Soy?

Ingeniero de Sistemas de la Universidad de Cartagena, especializado en desarrollo Fullstack y con experiencia en la implementación de soluciones tecnológicas orientadas a la mejora de procesos y experiencias digitales.

Mi Objetivo

Desarrollar soluciones web funcionales y atractivas que generen valor real, combinando tecnología, diseño y organización.

Mi Perfil

Apasionado por la creación de interfaces modernas con React, manejo de estados, APIs REST y metodologías ágiles como Scrum y Kanban. Me destaco por mi visión estratégica y trabajo en equipo.

Proyectos Principales

Pagina web CerdycomJf

Pagina web interactiva para la empresa CerdycomJf, que ofrece servicios de seguridad, rastreo, internet entre otros. La web permite a los usuarios solicitar servicios, ver información de contacto y conocer más sobre la empresa.

HTML • CSS • JavaScript • Bootstrap • React • Vite •Trello

Plataforma de Psicología

Software de gestión psicológica diseñado exclusivamente para el uso de las profesionales en psicología. Su objetivo principal es facilitar el almacenamiento, organización y seguimiento de la información clínica y emocional de los pacientes internos, específicamente docentes y trabajadores de cualquier empresa.

HTML • CSS • JavaScript • Bootstrap • Node.js • Docker-PostgreSQL • React • Vite •Trello

Software de nomina para empresa The English Easy way

Software de nómina desarrollado para la empresa The English Easy Way, que permite gestionar y automatizar el proceso de pago de salarios a los empleados y docentes de la institución, ademas de llevar el control de todos los procesos.

HTML • CSS • JavaScript • PHP • MySQL

Campus programas virtuales Universidad de cartagena

Campus virtual desarrollado para la Universidad de Cartagena, que permite a los estudiantes de programas virtuales y aspirates a conocer información relevante sobre los programas, inscribirse en cursos e interactuar con la comunidad académica.

HTML • JavaScript •CSS • React
